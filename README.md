## 실행 방법

```powershell
cd C:\Users\Win11Pro\Documents\Codex\2026-05-29\ai\kdt15-team4-project1
.\.venv\Scripts\python.exe -m flask --app app run --host 127.0.0.1 --port 5000
```

- 외부 중고거래 시세 조회
