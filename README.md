# 중고 경매 시세 서비스

중고거래 데이터를 기반으로 판매자와 구매자의 가격 판단을 돕는 경매형 중고거래 웹사이트입니다.

판매자는 물품을 등록할 때 저장된 중고거래 데이터를 기반으로 시세 그래프와 추천 시작가를 확인할 수 있고, 구매자는 경매 상세 화면에서 현재 입찰가가 적절한지 판단할 수 있습니다. 프로젝트의 핵심은 크롤링한 중고거래 데이터를 정리해 웹사이트 안에서 실질적인 가격 판단 자료로 활용하는 것입니다.

## 프로젝트 소개

중고거래에서는 같은 상품이라도 상태, 구성품, 거래 시점, 플랫폼에 따라 가격이 달라집니다. 이 때문에 판매자는 시작가를 정하기 어렵고, 구매자는 현재 가격이 적절한지 판단하기 어렵습니다.

이 프로젝트는 이러한 문제를 해결하기 위해 다음 기능을 제공합니다.

* 중고 물품 경매 등록
* 경매 목록 조회
* 경매 상세 조회
* 구매자 입찰
* 경매 마감 및 낙찰 결과 확인
* 관리자 검수 상태 관리
* 저장된 중고거래 데이터 기반 시세 그래프 제공
* 참고 시세, 시세 범위, 추천 시작가 제공

실제 결제, 회원가입, 로그인, 배송사 연동 기능은 프로젝트 범위에서 제외했습니다.

## 주요 기능

### 1. 경매 물품 등록

판매자는 상품명, 카테고리, 판매자명, 시작가, 입찰 단위, 마감 시간 등을 입력해 경매 물품을 등록할 수 있습니다.

등록 화면에서는 선택한 상품의 과거 중고거래 가격 데이터를 확인할 수 있어 시작가를 정할 때 참고할 수 있습니다.

### 2. 시세 그래프 확인

미리 수집하고 정리한 중고거래 데이터를 기반으로 상품별 가격 흐름을 그래프로 보여줍니다.

시세 데이터는 실시간 크롤링이 아니라 저장된 CSV와 SQLite 데이터를 기준으로 표시됩니다.

### 3. 입찰 기능

구매자는 경매 상세 화면에서 현재 가격과 시세 그래프를 확인한 뒤 입찰할 수 있습니다.

입찰가는 현재가와 입찰 단위를 기준으로 검증됩니다.

### 4. 낙찰 결과 확인

경매 마감 시간이 지나면 가장 높은 가격으로 입찰한 사용자가 낙찰자로 표시됩니다.

입찰자가 없는 경우에는 입찰 없이 마감된 상태로 처리됩니다.

### 5. 관리자 검수 흐름

낙찰 이후에는 판매자가 물품을 관리자에게 보내고, 관리자가 물품 검수 상태를 변경하는 흐름을 표현합니다.

검수 상태 예시는 다음과 같습니다.

* 판매자 발송 대기
* 관리자 수령 완료
* 검수 중
* 검수 통과
* 검수 보류
* 검수 실패
* 구매자 발송 완료

## 실행 방법

### 1. 저장소 클론

```bash
git clone https://github.com/vvybh/kdt15-team4-project1.git
cd kdt15-team4-project1
```

### 2. 가상환경 생성 및 실행

Windows PowerShell 기준입니다.

```powershell
python -m venv .venv
.\.venv\Scripts\activate
```

macOS 또는 Linux 기준입니다.

```bash
python3 -m venv .venv
source .venv/bin/activate
```

### 3. 패키지 설치

```bash
pip install -r requirements.txt
```

### 4. 데모 데이터 초기화

```bash
python scripts/reset_demo.py
```

### 5. Flask 서버 실행

```bash
python -m flask --app app run --host 127.0.0.1 --port 5000
```

또는 다음 명령어로도 실행할 수 있습니다.

```bash
python app.py
```

### 6. 브라우저 접속

```text
http://127.0.0.1:5000
```

## 데이터 처리 흐름

```text
중고거래 사이트 데이터 수집
        ↓
원본 데이터 저장
        ↓
상품명, 가격, 상태, 날짜 등 정리
        ↓
CSV 샘플 데이터 생성
        ↓
SQLite 데이터베이스 저장
        ↓
웹사이트에서 상품별 시세 그래프와 가격 참고 정보 제공
```

## 가격 참고 기준

상품별 수집 데이터를 기준으로 다음 정보를 계산합니다.

* 샘플 개수
* 참고 시세
* 시세 범위
* 추천 시작가
* 현재 입찰가의 시세 대비 판단

참고 시세는 수집된 가격 데이터의 중간값을 기준으로 계산합니다.

시세 범위는 데이터가 충분할 경우 낮은 쪽 25% 가격부터 높은 쪽 25% 가격까지를 기준으로 표시합니다.

## 크롤링 관련 주의사항

이 프로젝트는 학습 및 발표 목적의 팀 프로젝트입니다.

중고거래 데이터 수집은 공개된 페이지를 대상으로 하며, 제출 전 각 사이트의 이용약관과 robots.txt를 확인해야 합니다.

현재 웹사이트는 실시간 크롤링을 수행하지 않고, 미리 저장된 CSV와 SQLite 데이터를 기준으로 시세 정보를 표시합니다.
