# 검수 체크리스트

## 내용 검수

- [ ] 대상 독자가 명확한가?
- [ ] 문제를 먼저 설명했는가?
- [ ] 해결 방법이 결과 중심으로 설명되어 있는가?
- [ ] 어려운 용어가 쉽게 풀려 있는가?
- [ ] 근거 없는 숫자나 성과가 들어가지 않았는가?
- [ ] 확인되지 않은 내용은 `확인 필요`로 표시했는가?
- [ ] 남은 `확인 필요` 항목은 선택지로 정리했는가?
- [ ] 사용자가 하나를 고를 수 있게 선택 기준을 설명했는가?

## 데이터 수집 검수

- [ ] 크롤링 대상 사이트가 명확한가?
- [ ] 기술적으로 수집 가능한지 확인했는가?
- [ ] 정책상 수집이 허용되는지 확인했는가?
- [ ] 사이트 이용약관과 robots.txt를 최종 확인했는가?
- [ ] Open API를 쓴다면 사용 조건을 확인했는가?
- [ ] 수집할 데이터 항목이 정리되어 있는가?
- [ ] 개인정보를 수집하지 않는가?
- [ ] 수집 실패나 중복 데이터 처리 방법이 있는가?
- [ ] 크롤링 실패 시 기존 저장 데이터를 유지하는가?
- [ ] 웹사이트가 실시간 크롤링에 의존하지 않는가?

## 데이터 가공 검수

- [ ] 샘플 데이터 형식이 CSV 기준으로 정리되어 있는가?
- [ ] 웹사이트 조회 기준이 SQLite로 정리되어 있는가?
- [ ] CSV와 SQLite의 주요 컬럼이 서로 맞는가?
- [ ] 같은 물품을 묶는 기준이 있는가?
- [ ] 비슷한 조건을 비교하는 기준이 있는가?
- [ ] 가격 이상치를 처리하는 기준이 있는가?
- [ ] 물품 상태에 따른 가격 차이를 고려했는가?
- [ ] 데이터가 부족할 때 표시 기준이 있는가?
- [ ] 그래프에 데이터 기간과 출처를 표시했는가?
- [ ] 가격 기준이 시작가와 입찰 판단에 어떻게 쓰이는지 설명되어 있는가?

## 웹사이트 기능 검수

- [ ] 사용자가 경매 물품을 등록할 수 있는가?
- [ ] 사용자가 입찰할 수 있는가?
- [ ] 경매 마감 결과를 확인할 수 있는가?
- [ ] 낙찰 뒤 관리자 검수 상태를 확인할 수 있는가?
- [ ] 판매 등록 화면에서 시세 그래프를 볼 수 있는가?
- [ ] 경매 상세 화면에서 시세 그래프를 볼 수 있는가?
- [ ] 시세 그래프가 저장된 데이터 기준으로 표시되는가?
- [ ] 사용자가 물품을 검색할 수 있는가?
- [ ] 회원가입과 로그인 기능이 제외되어 있는가?
- [ ] 실제 결제 기능이 제외되어 있는가?
- [ ] 오류 상황에서 사용자가 이해할 수 있는 안내가 나오는가?

## 발표자료 검수

- [ ] 프로젝트 목표가 첫 부분에 명확히 보이는가?
- [ ] 경매 웹사이트가 주 목적임이 명확한가?
- [ ] 시세 그래프가 보조 기능으로 설명되어 있는가?
- [ ] 낙찰 뒤 관리자 검수 중개 방식이 설명되어 있는가?
- [ ] 데이터 수집, 데이터 가공, 웹사이트 기능이 연결되어 보이는가?
- [ ] 팀원별 역할이 명확한가?
- [ ] PPT가 10~15장 안에 들어가는가?
- [ ] 10분 안에 발표할 수 있는 분량인가?
- [ ] 시연 흐름이 준비되어 있는가?
- [ ] 한계와 개선 방향이 포함되어 있는가?

## 형식 검수

- [ ] 전체 구조가 자연스러운가?
- [ ] 제목만 읽어도 흐름이 보이는가?
- [ ] 문장이 너무 길지 않은가?
- [ ] 같은 표현이 불필요하게 반복되지 않는가?

## 산출물 검수

- [ ] 요청한 파일이 생성되었는가?
- [ ] 파일명이 이해하기 쉬운가?
- [ ] 결과물을 열어 확인했는가?
- [ ] 수정이 필요한 부분을 요약했는가?

## 노트북 시연 검수

- [ ] 발표 노트북에서 Flask 서버가 실행되는가?
- [ ] `http://127.0.0.1:5000` 접속이 되는가?
- [ ] 메인 화면이 웹사이트 대시보드 형태로 보이는가?
- [ ] 시세 그래프가 인터넷 연결 없이 표시되는가?
- [ ] 물품 등록, 입찰, 낙찰, 관리자 검수 화면을 순서대로 시연할 수 있는가?
- [ ] 실제 사이트 실시간 크롤링이 아닌 저장 데이터 기반임을 설명할 수 있는가?

## 확인 필요 처리 검수

- [ ] 확인이 필요한 결정을 한곳에 모았는가?
- [ ] 선택지는 2~3개로 정리했는가?
- [ ] 권장안을 먼저 제시했는가?
- [ ] 선택지마다 장단점을 짧게 설명했는가?
- [ ] 사용자가 고른 내용은 문서에 반영했는가?
