# 구현 태스크 체크리스트

## 목적

이 문서는 확정된 PRD를 바탕으로 코딩에 들어가기 위한 작업 목록을 정리한다.
각 태스크는 하나씩 구현하고 테스트할 수 있게 작게 나눈다.

## 기준

- 중심 기술: Python, HTML
- 웹 프레임워크: Flask
- 데이터 저장: SQLite
- 샘플 데이터 작성: CSV
- 그래프 표시: Chart.js
- 웹사이트는 실시간 크롤링을 하지 않음
- 웹사이트는 `data/app.db`의 저장 데이터만 조회
- 대표 상품 1개당 샘플 데이터는 5개 준비
- 회원 기능과 결제 기능은 제외

## 전체 개발 순서

1. 프로젝트 실행 뼈대를 만든다.
2. SQLite 데이터 구조를 만든다.
3. CSV 샘플 데이터를 SQLite에 넣는다.
4. 시세 계산과 그래프 데이터를 만든다.
5. 경매 목록, 등록, 상세 화면을 만든다.
6. 입찰과 낙찰 흐름을 만든다.
7. 관리자 검수 흐름을 만든다.
8. 검색과 오류 안내를 붙인다.
9. 크롤링 파트를 별도 작업으로 붙인다.
10. 발표 시연 흐름을 검수한다.
